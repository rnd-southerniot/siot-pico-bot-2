tasks/motor_task.py:
- Import uasyncio, lib.motor.Motor, lib.pid.PID, config
- Define module-level motor and PID instances using config constants (MOTOR_LEFT_A, MOTOR_LEFT_B, MOTOR_RIGHT_A, MOTOR_RIGHT_B, PID_KP, PID_KI, PID_KD)
- Define _target_rpm = {"left": 0.0, "right": 0.0} module-level state dict
- Define async def motor_pid_loop(): loops at 20Hz using await uasyncio.sleep_ms(1000 // 20). In this stub, measure actual dt with utime.ticks_diff rather than nominal interval (avoids Pitfall 4 from research). Print "motor tick" every 5 iterations. Wrap loop body in try/except Exception — on exception, brake both motors and re-raise (avoids Pitfall 2: gather stops on first exception, so log and brake first).
- Export: motor_pid_loop, get_target_rpm, set_target_rpm helper functions

tasks/sensor_task.py:
- Import uasyncio
- Define _sensor_state = {} module-level dict for shared sensor readings
- Define async def sensor_poll_loop(): loops at 10Hz using await uasyncio.sleep_ms(100). In stub, just updates _sensor_state with {"tick": counter} and prints "sensor tick" every 10 iterations. Wrap in try/except Exception — on exception log and continue (sensor failure must NOT kill motor task).
- Export: sensor_poll_loop, get_sensor_state

tasks/wifi_task.py:
- Import uasyncio
- Define async def wifi_placeholder(): loops with await uasyncio.sleep_ms(1000), prints "wifi placeholder" once per 10 iterations. Wrap in try/except. Comment: "Phase 2 replaces this coroutine with the HTTP server."
- Export: wifi_placeholder

  <name>Task 2: Rewrite main.py as async entry point + create gate1 + run_all</name>
  <files>main.py, gates/gate1_async_skeleton.py, gates/run_all.py</files>
  <action>
Rewrite main.py as the async entry point following the safe boot sequence from research (Pitfall 1: WDT must NOT start before calibration):
1. Import config, machine (Pin, I2C), lib.mpu6050.MPU6050
2. Print boot banner
3. Initialize I2C and MPU6050 — NO WDT yet
4. Call imu.calibrate_gyro_z(samples=200) with print "Calibrating IMU — keep robot still..."
5. Print "Calibration done"
6. Import WDT and start: wdt = WDT(timeout=8000); wdt.feed()
7. Define async main_async() that imports tasks and calls:
   uasyncio.gather(motor_pid_loop(), sensor_poll_loop(), wifi_placeholder(), wdt_feed_loop())
   where wdt_feed_loop is a simple coroutine defined inline that calls wdt.feed() every 4000ms
8. Call uasyncio.run(main_async())

Note: In this plan the WatchdogKeeper class does not exist yet (it's in Plan 04). Use inline wdt feed coroutine for now. Plan 04 will refactor this into safety/watchdog.py.

Create gates/gate1_async_skeleton.py — standalone proof-of-concept (does NOT import from tasks/, runs independently):
- Define three async tasks: task_a (prints "A {i}", sleep 100ms, 5 iterations), task_b (prints "B {i}", sleep 150ms, 5 iterations), task_c (prints "C {i}", sleep 200ms, 3 iterations)
- async main() gathers all three, prints "PASS: All 3 tasks completed without blocking"
- uasyncio.run(main())
- Expected output: interleaved A/B/C lines proving cooperative scheduling
