SIOT Pico Bot 2 Documentation
=============================

SIOT Pico Bot 2 is a Raspberry Pi Pico W robot kit with MicroPython firmware,
gate-based hardware validation, and a small browser control path over Wi-Fi.

This first-pass documentation site focuses on the current tracked repository:
how to get the robot running, what public API exists today, how the runtime is
organized, and how to use the on-device gates for validation.
